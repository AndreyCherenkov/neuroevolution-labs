# Нейроэволюционные вычисления (Весна, 2026)

Репозиторий содержит учебные материалы и практические работы по магистерской дисциплине **«Нейроэволюционные вычисления»
**. Курс посвящен изучению алгоритмов направленного эволюционного поиска для оптимизации структуры и весов искусственных
нейронных сетей, а также применению эволюционных стратегий в задачах искусственного интеллекта и управления.

---

## 📌 Обзор курса

В рамках дисциплины рассматриваются ключевые подходы к эволюции нейросетевых архитектур:

* **Фиксированные топологии:** Эволюция весовых коэффициентов (Conventional Neuroevolution).
* **Прямое кодирование и развивающиеся топологии:** Алгоритм NEAT (NeuroEvolution of Augmenting Topologies).
* **Непрямое (генеративное) кодирование:** HyperNEAT и использование паттернов (CPPN).

---

## 🛠 Стек технологий

* **Язык программирования:** Python 3.10+
* **Библиотеки нейроэволюции:** `neat-python`, `DEAP` (Distributed Evolutionary Algorithms in Python)
* **Глубокое обучение:** `PyTorch` / `TensorFlow`
* **Инструменты:** `NumPy`, `Matplotlib`, `MLflow` (визуализация траекторий и графиков сходимости)
